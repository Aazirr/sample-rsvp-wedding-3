# SunSpire Sample RSVP Showcase

A cinematic wedding invitation sample site for SunSpire Studios. This version is
intentionally static and showcase-friendly, with no database, auth, or `.env`
requirements.

## Getting Started

```bash
npm run dev
```

Open `http://localhost:3000`.

## Notes

- Editable content lives in `content/config.ts`
- Gallery images live in `public/gallery/`
- Project specs and progress tracking live in `docs/`
