# Creative Concept

## The vibe

Cinematic, dark, restrained. A 1920s smoke-and-whiskey world inspired by
*Peaky Blinders*, with crimson light leaks and a dramatic intro sequence.

## The cinematic intro

The intro flow remains:

```text
black screen -> match strike / lighter flare -> smoke curls ->
crest fades in: "By Order of the Hart Family" -> dissolve into hero
```

## The scenes

1. **Hero** - *Adrian Cole Hart & Serena Vale Bennett*, date, one line of copy,
   photoshoot hero image, red light leak bleeding from the edge.
2. **Our Story** - noir-flavored narrative of how they met.
3. **The Order of the Day** - date / time / ceremony + reception venue / map / timeline.
4. **The Dress Code** - 1920s formal / Peaky Blinders attire encouraged.
5. **The Gallery** - cinematic photoshoot frames.
6. **The Invitation (RSVP)** - personal accept/decline flow.
7. **Footer** - a closing line / quote.

## Copy & tone

- Hero: *"Adrian & Serena - a union, by order of the family."*
- RSVP accept: *"You're on the list. We'll see you there."*
- RSVP decline: *"Understood. You'll be missed."*
- Invalid link: *"This invitation could not be found."*
